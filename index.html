Plotsjs

Javascript library for plotting math functions or data arrays very similar to the 'plot' command found in Matlab or Scilab.

View On GitHub
Downloads:
  • ZIP
  • TAR

    Elegance

    The main concept is very simple : just show the values on the screen.

    Just like plotting a math function of real numbers.

    Pros

    • No need of other libraries
    • Simple and effective code
    • The snippet below works properly and is fully described on Github
    var data = [ ]
for (var k = 0; k < 10; k++) // randomly filled data array
 data.push( Math.random() )
plot (data, ' canvasId ' ) // this uses default options for colors, margins, etc

    If you're a Matlab(R) or Scilab user, you won't have trouble using this small library.

    More Examples

    There are a few examples that you can download and run in a single file (only one html). They cover randomness, sound, image and general electrical engineer subjects so as you to get in touch with the methods and properties.

    Unfortunately, due to security issues, some codes cannot be run from local hard disk, but demand a server. Nevertheless, it will be clearly mentioned and any XAMPP, Node.js, Joomla!, Webmatrix or whatever will let you run the code.

    Personally, I find it easier to run Node.js in Linux and Webmatrix in Windows.
